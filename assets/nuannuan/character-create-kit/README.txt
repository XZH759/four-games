裁切素材说明

本包共包含 279 个 PNG 裁切素材，按原图分为 5 组：
01_ui_controls：按钮、分页、单选框、复选框、开关、输入框等；
02_panels_cards：面板、卡片、列表项、弹窗等；
03_decorations：背景、舞台、平台、角花、星芒、挂饰、边框、丝带等；
04_character_reference：男女角色示例、发型、发色、眼睛、脸型、肤色、配饰、服装、体型及底部素材；
05_screen_regions：完整角色创建界面的主要区域裁切。

说明：
1. 当前为“保留原始背景”的精准裁切版，避免淡金色和浅紫色边缘在自动抠图时被误删。
2. 原图中部分元素彼此靠得很近，个别裁切可能仍保留少量相邻边框或装饰。
3. manifest.csv 记录了每个素材在原图中的裁切坐标。
4. previews 文件夹提供联系表预览，便于快速查找素材。
